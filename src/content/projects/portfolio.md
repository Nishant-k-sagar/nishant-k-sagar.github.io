## Portfolio website description and  Features


- **Astro** for static and lightweight site generation
- **Tailwind CSS** for styling
- **Content collections** for blog and projects
- **Monochrome UI** inspired by Kindle e-ink displays and Medium sleek UIs
- **Self-hosted fonts** for low delay and better privacy

## Tech Stack

- Astro 5.0
- Tailwind CSS 3.4
- TypeScript
- @fontsource/inter
- deployed on github pages
- Integrated github actions for CI/CD

## Design Goals

When I was designing this website, My primary goal was keeping it with minimal navigation friction. Easy to hop on other pages and things without feeling laggy and sluggish.

I have kept the design simple and sweet without any cluttered UI or styles. Easy to render for browsers without having any load on servers keeping it static.
